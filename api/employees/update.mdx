---
title: "Update Employee"
api: "PUT /employees/{id}"
description: "Update an employee's information"
---

## Authentication

This endpoint requires Bearer token authentication with one of the following roles:
- `super_admin`
- `hr_manager`

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the employee (UUID format)
</ParamField>

## Request Body

All fields are optional. Only include fields you want to update.

<ParamField body="phone" type="string">
  Employee phone number
</ParamField>

<ParamField body="first_name" type="string">
  Employee first name
</ParamField>

<ParamField body="last_name" type="string">
  Employee last name
</ParamField>

<ParamField body="date_of_birth" type="string">
  Employee date of birth (format: YYYY-MM-DD)
</ParamField>

<ParamField body="department_id" type="string">
  Department UUID
</ParamField>

<ParamField body="designation_id" type="string">
  Designation UUID
</ParamField>

<ParamField body="level_id" type="string">
  Level UUID
</ParamField>

<ParamField body="manager_id" type="string">
  Manager UUID
</ParamField>

<ParamField body="status" type="string">
  Employee status. Must be one of:
  - `active`
  - `inactive`
  - `on_leave`
  - `terminated`
  - `probation`
</ParamField>

<ParamField body="gender" type="string">
  Employee gender
</ParamField>

<ParamField body="address" type="string">
  Employee address
</ParamField>

<ParamField body="emergency_contact_name" type="string">
  Emergency contact name
</ParamField>

<ParamField body="emergency_contact_phone" type="string">
  Emergency contact phone number
</ParamField>

<ParamField body="profile_image_url" type="string">
  Profile image URL
</ParamField>

<ParamField body="termination_date" type="string">
  Termination date (format: YYYY-MM-DD)
</ParamField>

<RequestExample>

```bash cURL (PUT)
curl -X PUT https://api.companyflow.com/employees/emp-uuid-here \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+1234567899",
    "status": "on_leave",
    "address": "456 New Street, City, State"
  }'
```

```bash cURL (PATCH)
curl -X PATCH https://api.companyflow.com/employees/emp-uuid-here \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "department_id": "new-dept-uuid",
    "designation_id": "new-desig-uuid"
  }'
```

</RequestExample>
