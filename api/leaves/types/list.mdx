---
title: 'List Leave Types'
api: 'GET /companies/{company_id}/leave-types'
description: 'Get paginated list of leave types for a company'
---

Retrieve a list of all leave types configured for a company with support for pagination, filtering, and search.

## Authentication

Requires authentication with Bearer token. Available to:
- **Super Admin**
- **HR Manager**
- **Employee**
- **Manager**

## Path Parameters

<ParamField path="company_id" type="string" required>
  The unique identifier of the company (UUID format)
  
  **Example:** `"123e4567-e89b-12d3-a456-426614174000"`
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Page number for pagination (starts at 1)
  
  **Default:** `1`
</ParamField>

<ParamField query="pageSize" type="integer">
  Number of items per page
  
  **Default:** `10`
</ParamField>

<ParamField query="status" type="string">
  Filter by leave type status: `active` or `inactive`
  
  **Example:** `"active"`
</ParamField>

<ParamField query="search" type="string">
  Search by name or code (case-insensitive)
  
  **Example:** `"annual"`
</ParamField>

## Filtering and Search

<Tip>
  Combine multiple query parameters to refine your results:
  
  - Use `status=active` to see only active leave types
  - Use `search=sick` to find leave types containing "sick" in name or code
  - Use pagination parameters to control result size
</Tip>
